# project-init

> package for custom project initialization

## Install
```
npm i git://github.com/max-norin/project-init.git#dev
```

## Usage
```javascript
// .eslintrc.js

/**
 * ESLint Configuration
 */
module.exports = {
  extends: ['./node_modules/project-init/templates/nuxt/.eslintrc.js']
}
```
