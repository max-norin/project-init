# project-init

> package for custom project initialization

## Install
```
npm i git://github.com/max-norin/project-init.git#dev
```

## Usage
```javascript
// .eslintrc.js

/**
 * ESLint Configuration
 */
module.exports = {
  extends: ['project-init/templates/nuxt/.eslintrc.js'],
}
```
